Google Setup Guide

To use this app for free, you must create your own "Google Project" (takes 5 mins).

  1. Go to Google Cloud Console on a PC.
  2. Create a New Project.
  3. Search for "Gmail API" and click Enable.
  4. Go to OAuth Consent Screen:
    • Set User Type to External.
    • Add https://www.googleapis.com/auth/gmail.readonly to Scopes.
    • Add your own email to Test Users.
  5. Go to Credentials > Create Credentials > OAuth Client ID:
    • Application Type: Web Application.
    • Authorized Redirect URI: https://mattwhite7982.github.io/pebble-auth/
  6. Copy the Client ID and paste it below.

Link Your Account

Note: You will see a "Google hasn't verified this app" warning. Click Advanced > Go to [App] (unsafe) to proceed.
