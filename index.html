Google Setup Guide

To use this tracker for free, follow these steps in the Google Cloud Console:

  1. Create a New Project.
  2. Enable the Gmail API.
  3. On the OAuth Consent Screen:
    • Set to External and add your email to Test Users.
    • Add scope: https://www.googleapis.com/auth/gmail.readonly.
  4. Create OAuth Client ID (Web App):
    • Redirect URI: https://mattwhite7982.github.io/pebble-auth/.
